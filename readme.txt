Question #1 A: The possible ways to apply a CSS style to an HTML page are Inline, embedded/internal/style, and external. The precedence order is
inline being highest priority and overriding other applications, then internal, then external (external css file).

Question #1 B: Margin is the space between elements in the html file and padding is the space between the content and the border of the element.

Question #2 A: "!==" is a comparison operator that checks whether the value of two objects is not the same, "!===" is a comparison operator that
checks whether the value and TYPE of two objects is not the same.
